# Portfolio Images

This folder contains all images for your digital portfolio.

## Adding Your Artwork

1. Place your image files directly in this folder
2. Supported formats: JPG, PNG, WebP, GIF
3. Use descriptive filenames (e.g., `sunset-landscape.jpg`, `portrait-sketch.png`)

## Important Notes

- **Image names in portfolio.json must match exactly** (case-sensitive)
- The sample data in `src/data/portfolio.json` references example images:
  - `sunset-landscape.jpg`
  - `portrait-sketch.jpg`
  - `abstract-composition.jpg`
  - `floral-study.jpg`
  - `urban-skyline.jpg`

## Recommendations

- **Resolution:** At least 1200px on the longest side
- **File size:** Under 5MB per image (Next.js will optimize them)
- **Format:** JPG for photographs, PNG for graphics with transparency
- **Naming:** Use lowercase with hyphens (e.g., `my-artwork.jpg`)

## Image Optimization

Next.js automatically optimizes your images:
- Converts to modern formats (WebP/AVIF)
- Responsive sizing for different screen sizes
- Lazy loading for better performance
- No need to manually resize or optimize
